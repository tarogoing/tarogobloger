# 跨境汇款秒到账、千一手续费？

### 留学生家长怎么向在国外的孩子打钱？一般有这么几种方法：
**银行网点汇款**。工行为例，带证件材料，柜台填表，0.1%手续费，最低50元/笔，另加收电讯费，到账时间不确定；
**电子银行转账**，0.6%/手续费，最低60元/笔，另加收电讯费、邮费。
**国际汇款公司**。西联为例，手续费15美元起，数分钟到账。
**转人民币**。孩子在支持银联的国外银行ATM取外币，收取提现手续费。
**托朋友“人肉”带**现金或汇票……
上述办法均涉及结售汇，需承担双边点差。
TSF CEO高一萌向36氪介绍，自己在国外生活时还有更“民间”、更省钱的“汇款”方式：高一萌的父母在国内向朋友A转人民币，朋友A在国外的孩子给高一萌等值外币，done！
我们暂且把这种“去中心”的方式称为”民间交换”。细想一下，走这条“捷径”需要几个条件：刚好需求（金额、币种、四方参与者所在国家）匹配，刚好找得到合适的人，刚好四方信任彼此。前两个“刚好”可以由互联网解决，而“刚好信任”则要借助区块链技术。
高一萌先从加拿大开始做调研和测算：加拿大华人移民多，大多在国内有亲戚，经常汇款回国；留学生基数大，国内家长汇学费、生活费的频率较高；代购需求同样旺盛。当总用户达10万量级，可以实现需求秒级匹配。
因此，高一萌于3月组建团队，创办TSF（Easytransfer），希望通过区块链连接有跨境汇款需求的人群，拆分或整合两端的换汇、转账需求，用智能合约自动执行“民间交换”，预防欺诈风险，让资金以更低的成本去信任、秒到账。
产品上线后，用户只需实名注册，绑定银行账户，发布需求（包括手持币种，收款币种及金额，所在国家等），APP将向收款地区用户推送，有人接单即生成智能合约。
在后端，资金经由数字钱包完成划转，底层的数字货币与美元锚定，无交易功能，相对价格稳定，仅作为各法币币种之间的介质。
由于币种不变、资金也不出境，所以不涉及洗钱，也无需对接外管局的结售汇或跨境汇款系统。
每人每年可汇款1万美元，超过1万美元需提交补充资料。
每笔交易双向收取0.1%手续费，用户用**COIN（TSF的token）支付。除了购买COIN，用户还可通过邀请新用户等活动获取COIN（相当于活动期免费汇款）**。同时，TSF还在与加拿大通用消费积分公司Air Miles合作，Air Miles的会员加油、买机票等均可获得COIN。前文也提到，需求快速匹配的基础是用户足够多，所以，TSF也会通过海外的便民网站和社群传播获客。
做深留学生市场后，TSF希望切向境外游用户，与热门旅游国家的银行合作，用户可在国内“存入”人民币，到了海外机场，直接在柜台领取“存好”外币的国外银行卡。
据介绍，TSF正在进行首轮融资，计划融资额90万美元，释放20%股份。虽然发行COIN，但高一萌更希望token只是获客手段，而非融资工具。
团队共7人，有连续创业和共事经历，其中2人出身技术背景。高一萌，曾创办现货对冲交易平台北京泰合鑫和北京鑫时代，大宗商品交易平台湖北九汇交易所，宠物保险公司“一直宠”。联合创始人裴飞，2016年起专注金融科技和加密货币投资，曾任Consensusu Fintech Group联创，曾供职蚂蚁金服，从事金融科技创新研究。其他成员来自赛伯乐、七彩单车、网易等，有丰富的区块链底层框架搭建、智能合约相关经验。
团队自3月起开发，考虑到用户基础、开发者习惯和智能合约成熟度，选择以太坊作为底层，预计7月上线APP“Transfer”，未来将开发自己的公有链，支撑应用层。
在跨境汇款业务领域，36氪曾介绍过Ripple、OKCoin旗下的OKLink、YeeCall、PeerTransfer、TransferTo、Circle Pay、EMQ、可零可零等。其中不乏有大量用户基础的公司，但能跳出“中心化”金融机构的既存规则，通过关联者实现C端资金不出境互换的并不多。
