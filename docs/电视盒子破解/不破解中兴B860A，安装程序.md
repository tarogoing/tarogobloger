# 不破解中兴B860A，安装程序

本人小白一个，使用的是深圳电信的IPTV，盒子是B860A，一开始学习按照坛子里的方法安装第三方软件均未成功，后来尝试着在手机上安装“悟空遥控器”，抱着试一试的态度，从手机推送软件到盒子安装，没想到，居然成功了！是的，真的成功了！！
 方法如下：
1. 将B860A的界面设置为“时尚版”后重启；
2. 在手机上安装“悟空遥控器”并打开，确保手机与盒子处于同一个WIFI网络；
3. 经过以上两步，应该能够在手机上看到盒子的IP地址设备了，连接这个设备；
4. 在“悟空遥控器”中搜索“当贝市场”应用，并推送到盒子上安装。

OK，大功告成！其它需要安装什么应用，打开“当贝市场”安装即可。
 第三方应用安装后，打开时使用“悟空遥控器”来操作。使用软件过程中盒子的遥控器完成能够操作，只是菜单键仍然要用“悟空遥控器”，当然，大家也可以安装“当贝桌面”或者其它桌面应用试试。
 重要提醒：如果你连接的是电信的IPTV有线网络，那么安装的第三方应用是不能联网的，因而不能观看节目，只有盒子自带的应用能够使用。所以，需要切换成WIFI网络或者与路由器连接的有线网络，这样，既不影响自带软件的使用，也可以尽情享受第三方应用的舒爽感觉！真正让盒子能够物尽其用。
 好东西无需隐藏，祝大家成功！
