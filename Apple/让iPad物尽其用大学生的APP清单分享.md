# 一、学习类APP

工欲善其事，必先利其器。要想让你手里的iPad和Apple Pencil在学习上发挥作用，势必需要一系列笔记APP的支持。

![让iPad物尽其用：大学生的APP清单分享](../_resources/43b8d2d803af4a7787c189a76bc61f1c_77f15ce2ebf34ffa8.jpg)

# 1、日常笔记/标注：GoodNotes 5

俗话说：好记性不如烂笔头。对于习惯了手写记笔记的我来说，iPad的最重要用途就是记笔记。电子笔记比起传统纸笔，优势明显：首先，可以使用各种颜色的字体，可以用荧光笔标记重点，“一笔搞定”，而不需要像传统纸笔那样带一堆彩色笔；其次，一个APP可以装很多很多的笔记，携带更方便；再次就是笔记APP的搜索功能，可以快速找到你要的功能；最后，电子笔记可以很方便地插入图表，比如PPT，书本插图，类似于剪报。

在多款记笔记的APP中，我用得最顺手的就是Notability和GoodNotes 5，这两款APP可以说各有千秋，不过我最终还是选择了GoodNotes 5。目前我主要用它来进行日常的笔记和短论文的阅读。

![让iPad物尽其用：大学生的APP清单分享](../_resources/5a576cd7276e4d2fa2ce8ca9d41efd13_c79fb06505e747119.jpg)

▲Notability

这款APP最大的优点就是“无限笔记页面”，哗啦哗啦一直往下拉，非常适合做草稿纸，此外还可以在笔记内直接嵌入录音；不过模板形式较为单一，写字感觉不如GoodNote 5，而且不支持Office文档（doc，ppt等）的标注。

![让iPad物尽其用：大学生的APP清单分享](../_resources/5311149d3a554ab9aa96f164267742ef_4e9c0a8a3e6040ab9.jpg)

▲GoodNotes 5笔记界面

我之所以最终选择了GoodNotes 5，最主要还是因为良好的写字手感。这款APP是我认为的写字感觉最接近于真实纸笔的APP。

看PDF的课本和论文，我开始用的是PDF Expert，后来还是觉得GoodNotes 5更好用：

![让iPad物尽其用：大学生的APP清单分享](../_resources/39837a956d9e4e5ab4457d04be5d7878_8d2b1a3c4b7c469a9.jpg)

▲GoodNotes 5 标注PDF

在分屏功能上，GoodNotes 5的操作比Notability要稍微繁琐一些：点击Dock栏的APP图标，在后台界面右上角点击“+”号，即可开启另一个GoodNotes窗口，拖动Dock栏的图标即可分屏：

![让iPad物尽其用：大学生的APP清单分享](../_resources/ed5a0ee701c44e27a6102aa7d10f69d3_7de38af8462d49f59.jpg)

▲GoodNotes 5分屏

一边看书一边记笔记问题也不大，不过如果经常分屏的话，12.9寸比较适合，11寸我觉得有一点小了。

![让iPad物尽其用：大学生的APP清单分享](../_resources/e650135eea444610bf26f963353a9fc6_7e254dcdcaf54e37a.jpg)

▲GoodNotes 5分屏示例

这样分屏的好处在于，如果看到书里有什么图表要摘录到笔记，直接用APP自带的“套索”工具，复制粘贴就行。

GoodNotes 5除了自带的众多模板之外，还可以去淘宝或者网上找一些第三方模板的资源，可以拿来做手帐。

![让iPad物尽其用：大学生的APP清单分享](../_resources/0a02554317c649ddafe72ee4e137b52b_56f8b614b55d4e3eb.jpg)

▲GoodNotes 5第三方模板

最后就是笔记的同步问题，GoodNotes 5支持iCloud同步笔记，可以同步到其它平台，以后更换了iPad也不用担心笔记丢失。

个人觉得，GoodNotes 5是一款非常不错的“全能”笔记APP，适合日常笔记，课件、论文标注。如果经常使用，建议购买正版，以免重要笔记丢失。

# 2、“拆解”书本：MarginNote 3

在研究生阶段的学习中，我也接触了MarginNote 3，这可以说是一款全新的笔记APP，可以将书本整理成思维导图，更方便记忆。目前我主要拿它阅读专业课本和综述类论文。

我个人看书，无论是课本、论文，都喜欢做笔记，加深印象；但对于篇幅较长的书来说，记笔记虽然可以加深印象，但是却不能将知识“分门别类”，复习起来缺乏条理性。MarginNote 3可以将PDF制作成思维导图，将书本“拆解”，有利于整理和归类。

![让iPad物尽其用：大学生的APP清单分享](../_resources/08be2d1e07a34ffa9a890cf75335e2a6_9dc933a74295484a9.jpg)

▲MarginNote 3界面

它的界面分成左右两侧，我们直接用“手”这个工具，拖动你想建立思维导图的文字，即可生成思维导图，拖动即可修改思维导图的分支关系。

![让iPad物尽其用：大学生的APP清单分享](../_resources/17f3a3cb6dbb45b494e88ccd09123012_2875b43618d743cba.jpg)

▲思维导图每一个节点都可以自定义，可以用不同颜色表示，还可以用Apple pencil标注，有问题还可以点击“研究”按钮，目前可以链接到维基百科、知乎和谷歌翻译。

在同一个笔记本下，还可以放置多个思维导图，你可以把同一类型的多篇文章全部归类到一个笔记本中，来自不同文档的摘录也可以放到同一个图中，还可以用虚线建立联系。

![让iPad物尽其用：大学生的APP清单分享](../_resources/e41f7b8af50340c8acabdf29b49741cf_26641e12bb1f46a29.jpg)

▲多选某一些节点，加入卡片组

加入卡片组有什么作用呢？这就不得不提及MarginNote 3的另外一大功能——复习。

![让iPad物尽其用：大学生的APP清单分享](../_resources/5d10e0512f5b48e6bf4792dd3cf9af4d_f742ff78542442b99.jpg)

▲复习模式

每一个思维导图节点，都是一张卡片，卡片的正面就是标题，背面就是你框选的内容，点击“▶︎”符号，即可进入这样一个页面：

![让iPad物尽其用：大学生的APP清单分享](../_resources/de945981c4fe41569cd393a7b961749e_0edc2b7493a64c689.jpg)

▲考试模式

点击一下，答案就出来了。我们可以利用这个功能，对一些需要反复记忆的东西，比如公式、单词等，进行多次自测，加深记忆。

思维导图现在已经成了大家广泛使用的一个工具，用来对知识进行归类整理，让复习更具有条理性。而MarginNote 3，则免去了我们手动制作思维导图的麻烦，复习模式更是可以对一些知识点实现“重点突破”，确实是学习的又一大“神器”。

# 3、转换工具人：CAJ云阅读

中国知网出品的APP，可以阅读知网CAJ格式的论文。其实它不仅可以阅读，还可以一键转换为PDF：

![让iPad物尽其用：大学生的APP清单分享](../_resources/743ddd20d71c49068d3686bf68e35982_758c7f52f4a5497aa.jpg)

▲一键转换功能

直接右滑就可以看到菜单。转化为PDF之后，GoodNotes 5、MarginNote3就能轮番上阵了，不需要用那个功能很少、界面简陋的云阅读来看论文。

所以我把它当“工具人”用。

# 4、查词神器：欧路词典

适配iPadOS的词典类APP非常多，也有一些经典的词典可以选择：

![让iPad物尽其用：大学生的APP清单分享](../_resources/cf619c6ecd9847a98f8c9b9d5aae278e_1f48cb8542b74861a.jpg)

▲AppStore的词典合辑

但是今天推荐的APP是欧路词典，一款非常适合论文阅读的词典APP。在不少学科英文文献的阅读中，常常会涉及不少专业词汇，GoodNotes也好，MarginNote也罢，都能调用系统自带的词典，不过在专业词汇面前，自带词典经常抓瞎。而欧路词典完美地解决了我的需求，面对化学类专业词汇游刃有余。

![让iPad物尽其用：大学生的APP清单分享](../_resources/88ce1c8a771c4dd7ba1ab291a8e4c8c0_a17c2a0a7dfe4941b.jpg)

▲分屏查词

我太喜欢这个功能了，只需要点击“拷贝”，欧路词典就能自动帮你查词，不需要手动复制粘贴。实测专业词汇的查词准确率非常高，在线的“联网翻译”也相当精准。

此外，欧路词典还可以背单词、甚至全文翻译。这些功能因为我平时没怎么用不太熟，就不在这里讲了，有需要的可以自己去了解一下。

# 5、本地视频播放器：nplayer

nplayer是一款非常好用的本地视频播放器，支持“画中画”模式。

# 二、工作类APP

说到“生产力”，大家肯定会想到Office，在iPad平台上也有不少不错的办公软件，配合一把蓝牙键盘，可以临时地充当笔记本用，写一些东西。

![让iPad物尽其用：大学生的APP清单分享](../_resources/2fb9a6e283714db18e4bafcf40b6fcd5_59efcdc35bc44c97b.jpg)

# 1、支持markdown格式的编辑器：熊掌记

对我们这种需要“一文多发”的作者来说，markdown堪称“神器”。它可以通过转换小工具，直接转化为微信公众号所需要的排版样式；还可以搭配图床使用，无论在哪发布都不会丢图，省去了手动上传图片的烦恼；部分网站（比如知乎）更是支持一键导入.md格式的文档，非常便捷。

iPad端的markdown编辑器，我选择了熊掌记，它界面美观，操作方便，也支持markdown语法。

![让iPad物尽其用：大学生的APP清单分享](../_resources/ea858aee09584a83b9b8221145ecf529_89fd20a4462e42f98.jpg)

▲熊掌记界面

还可以插入待办事项、代码块、图片，甚至可以用Apple Pencil手写，再插入到笔记当中。有了熊掌记，在iPad上写公众号的推文，也不是什么难事儿。

# 2、思维导图：幕布

iPadOS平台也有不少好用的思维导图APP，幕布就是其中一个。和传统的思维导图（比如XMind）不同，它默认就是大纲模式，用起来更顺手（当然XMind也可以切换为大纲模式，只不过它默认是脑图模式）。

![让iPad物尽其用：大学生的APP清单分享](../_resources/cea5e8aa0905471db1ee371f851ced05_fdf2be056b934c28a.jpg)

▲幕布界面

前面提到了，思维导图可以让思路更加有条理化，不仅在学习中如此，在工作中也是这样，这篇文章也是先用幕布APP做好思维导图，再组织内容的。

iPad自带了苹果的Office三件套——Pages、Numbers和Keynote。Office套件也是我经常要使用的，更是“生产力”的标配。

![让iPad物尽其用：大学生的APP清单分享](../_resources/325eac2a4d684efaa3aeac6007decefd_a8774d0116ca40fd9.jpg)

▲Pages自带模板

Pages常见的功能都有，自带的模板也不少，支持保存为doc格式。

![让iPad物尽其用：大学生的APP清单分享](../_resources/6882857b2e574d61b4fd6f75531cd82a_dfbc37ea3ffc4dab8.jpg)

▲Numbers

Numbers自带了一些简单的公式，如果只是制作一些简单的表格，Numbers完全可以胜任。

# 4、PDF签名神器：PDF Expert

最开始我用这款APP来标注PDF，后来实在不习惯它的写字触感，换成了GoodNotes 5。不过我依旧保留了这款APP，主要是它的签名功能太好用了。

疫情期间，很多时候需要远程办公，PDF Expert可以设置好签名，直接插入到PDF文档中：

# 5、简单的视频剪辑：剪映

iPadOS上最强大的视频剪辑软件应该就是LumaFusion，这是一款专业级别的视频剪辑APP，B站上不少up主也有演示过。不过对于简单的视频来说，有点“大炮打蚊子”了，系统自带的iMovie固然可以胜任，但是模板有点少，对比了几款APP之后，我选择了剪映。

![让iPad物尽其用：大学生的APP清单分享](../_resources/c882b1ba74a4442e861f6c4e66c333ba_453fed4b4d4c4d6f9.jpg)

剪映比较适合短视频剪辑，尤其适合抖音。上面有很多模板，可以轻松剪出“网红视频”，本文开头的视频，也是用iPad上的剪映制作的。

不过剪映不能横屏使用，对于习惯横屏用iPad的我来说有点不适应。

# 6、随手涂鸦：概念画板

人的思维有时候是很跳跃、很抽象的，灵感可能稍纵即逝，而涂鸦，是保留灵感的最佳方式。iPadOS平台上也有不少专业的绘画软件，比如procreat，不过对于我这种“灵魂画手”来说，概念画板够用了。

![让iPad物尽其用：大学生的APP清单分享](../_resources/1fed939675e545178f41d95bec704447_a1fc3604d5404c50a.jpg)

▲概念画板界面

各种笔刷基本齐全，色轮里可选的色彩也比较多，会画画的完全可以用这款APP创作出一幅相当不错的画作，对于我等没学过画的人来说，这款APP也是绝佳的涂鸦工具，可以随手记录下自己的一些想法。

# 六、写在最后

去年那篇文章的标题是“iPad+Apple Pencil是不是大学生学习的好伙伴？”，一年过去了，我的答案是：当然是。我周围也有越来越多的同学开始使用iPad+Apple pencil来学习。

不过，我觉得还是看个人的自制力，有些人自制力很差，游戏卸了下，下了又卸，如果你是这种人，我建议你采用更强硬的控制手段，不要买iPad学习了，直接一支笔，一本草稿本，几本书，这种最传统的学习方式，兴许是最有利的；而对于自制力较强，目标明确，计划合理的人群，iPad自然会成为你学习的好帮手，这个不需要顾虑（我也相信你不会顾虑）。

今天一共推荐了18款APP，都是我个人常用，觉得不错的应用，推荐给大家，权当抛砖引玉。昨天晚上和几个老朋友聚会，刚巧大家都有iPad，职业也不同：医学生会用Altas看人体解剖图；室内设计师用CAD看图王看图纸；教师则用Office看课件，并做修改。由于我自己所学的专业也没有什么特别的APP，所以今天推荐的这些，适合大多数大学生/研究生使用。

如果大家的iPad有一些不错的APP，欢迎留言分享。让iPad物尽其用，而不是单纯的娱乐平板。

# 真正的结尾

虽然我不认为现在的iPad可以替代Mac/PC，但随着时间的推移，还真的不好说。现在的iPad Pro，接上一个扩展坞，再配上妙控键盘，已经可以完成不少原来只能在电脑上完成的任务。尽管iPad一开始的定位是娱乐平板，但苹果近年来一再发掘iPad的生产力潜力，iPad OS现在已经拥有相当完善的文件管理系统，跟PC/Mac完全一致的网页浏览体验，在可见的将来，iPad或许能拥有媲美PC/Mac的生产力。

在我看来，iPad对于一个大学生来讲，是手机、PC之后的第三设备，是大学期间学习娱乐的好伙伴。这篇文章的写作目的，更多地是让大家充分利用手里的iPad。
