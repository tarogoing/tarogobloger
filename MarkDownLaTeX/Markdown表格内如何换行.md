用HTML代码中的`<br>`

如下所示，要在“身高”换行后加“体重”，可以在本行末尾输入`<br>`，再输入“体重”，则马上换行显示。

**源码：**

```
| 姓名 | 特征        |
| ---- | ----------- |
| 小王 | 身高：175cm |

| 姓名 | 特征                     |
| ---- | ------------------------ |
| 小王 | 身高：175cm<br>体重:45kg |

```

**显示效果：**

| 姓名  | 特征  |
| --- | --- |
| 小王  | 身高：175cm |

| 姓名  | 特征  |
| --- | --- |
| 小王  | 身高：175cm  <br>体重:45kg |
